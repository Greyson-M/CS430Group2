# Front

## Dependencies

- [node.js](https://nodejs.org/en/download)
- [React](https://reactjs.org/)
- react-scripts
- [react-router-dom](https://reactrouter.com/)

## How to Install

1. Clone the repository.
2. Navigate to the front-end directory:

   ```bash
   cd Front/front-end-app
   ```

3. Install Node.js dependencies:

   ```bash
   npm install
   <!-- npm install react-router-dom -->
   ```

## How to Run

1. In your terminal, navigate to the project root or `front-app` folder.
2. Start the React development server:

   ```bash
   npm run dev
   ```

## Troubleshooting

- If the app doesn't run, ensure Node.js is installed with all dependencies.
- You may need to resolve permissions issues in your terminal or PowerShell.
- If you encounter issues with missing packages, run `npm install` again.
